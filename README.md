## 🛠 データの形式について
全てのデータは、外部アプリやシステムから利用しやすいよう [JSON形式](./keikanwan_tests.json) で管理されています。

### データの追加方法
1. 各カテゴリーの `.json` ファイルに新しい項目を追記します。
2. この `README.md` の一覧テーブルに項目を追加します。
